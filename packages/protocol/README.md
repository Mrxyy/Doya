# @getdoya/protocol

Shared Doya protocol schemas, codecs, and wire types.

## Stability

This package is public so Doya's published packages can depend on it cleanly.
It is not a stable public API yet.

Schemas, exports, wire helpers, and types may change or disappear in any release
without advance notice. Use it outside Doya at your own risk until the package
is explicitly documented as stable.

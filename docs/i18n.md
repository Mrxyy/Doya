# i18n

Paseo app 的 i18n 入口在 `packages/app/src/i18n/`。

- 当前支持 `en` 和 `zh`。
- 用户偏好存储在 app settings 的 `language` 字段，取值为 `system`、`en`、`zh`。
- React 组件使用 `useI18n()`，非 React 边界使用 `translateNow()`。
- 新增用户可见文案必须进入 `translations.ts`，不要在账号、工作区、设置入口继续写硬编码中文。
- 老界面可以逐步迁移；迁移时优先处理新多租户流程、设置入口、错误提示和按钮。

扫描未迁移中文：

```bash
rg '"[^"]*[\p{Han}][^"]*"|`[^`]*[\p{Han}][^`]*`|>[[:space:]]*[^<]*[\p{Han}]' packages/app/src -n
```

迁移复查不要只扫中文。英文 UI 文案常藏在 toast、accessibility label、panel descriptor、dialog title/message、picker label 里。运行时源码可先用这些热点扫描：

```bash
rg -n '(Alert\.alert\(|toast\.show\(|toast\.error\(|toast\.copied\(|placeholder=|accessibilityLabel=|title=|message=|label=|description=|children=)\{?\s*["`]([A-Z][A-Za-z][^"`]{2,})' packages/app/src -g '*.ts' -g '*.tsx' -g '!**/*.test.ts' -g '!**/*.test.tsx' -g '!**/i18n/translations.ts'

rg -n '"[^"]*[\p{Script=Han}][^"]*"|`[^`]*[\p{Script=Han}][^`]*`|>[[:space:]]*[^<]*[\p{Script=Han}]' packages/app/src -g '*.ts' -g '*.tsx' -g '!**/*.test.ts' -g '!**/*.test.tsx' -g '!**/translations.ts' -g '!**/acp-provider-icons.ts'
```

For JSX text and object descriptors, use an AST scan rather than relying only on regex. Exclude tests, `translations.ts`, assets, and test utilities; fixed file names such as generated attachment names are not UI copy.

翻译 key 使用扁平命名：`domain.surface.intent`，例如 `openProject.accountAuth.title`。
